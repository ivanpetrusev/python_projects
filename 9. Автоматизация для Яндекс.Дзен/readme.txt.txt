/// Рекомендации по открытию файлов, расположенных в архиве

_________________________________________________________
База данных хранится в файле: zen.dump

Порядок развертывания базы
Создание базу данных командой:
-createdb zen --encoding='utf-8'

Войдите в консоль PostgreSQL и выполните команды:
\c zen
CREATE USER my_user WITH ENCRYPTED PASSWORD 'my_user_password';
\q

Загрузите в базу данных данные из бэкап-файла:
pg_restore -d zen zen.dump

Войдите в консоль PostgreSQL и выполните команды:
\c zen
\dt
select count(*) from log_raw;

В таблице log_raw:
- сохранены события в промежутке между 18:00 и 19:00 24 сентября 2019 года (60 минут);
- всего 6 возрастных групп;
- встречаются события 3 типов;
- есть 25 тем карточек и 26 тем источников.

_________________________________________________________
Пайплайн файл: zen_pipeline.py

В командной строке установите sqlalchemy:
python -m pip install sqlalchemy
python -m pip install psycopg2
Первая команда установит библиотеку sqlalchemy, а вторая — psycopg2.

В командной строке перейдите в папку, в которой сохранён скрипт пайплайна. 
Если вы работаете с Linux или Ubuntu, выполните команду:
python3 zen_pipeline.py
А в Windows 7 или MacOS — такую команду:
python zen_pipeline.py

Вызов скрипта пайплайна диапазона тестовых данных через командную строку: 
python zen_pipeline.py --start_dt='2019-09-24 18:00:00' --end_dt='2019-09-24 19:00:00'
	
_________________________________________________________
Дэшборд хранится в файле: zen_dash.py

Linux и Ubuntu
В командной строке выполните команду установки dash:
sudo pip3 install dash==1.6.1
Windows 7
В командной строке выполните команду установки dash:
python -m pip install dash==1.6.1

Затем в командной строке перейдите в папку, где вы сохранили 
zen_dash.py. Если вы работаете с Linux или Ubuntu, выполните команду:
python3 zen_dash.py

А в Windows или MacOS:
python zen_dash.py

Что посмотреть дэшборд -скопируйте строку "http://127.0.0.1:8050/" (пример) и откройте эту ссылку в браузере.
